# quick-metadata
small program to get specific metadata from image files


## Installation
- Download latest Release_v*.zip
- unzip programm folder
- run install.ps1 as admin


note: 
if windows blocks the ps1 file, run this command once in the powershell console as admin:

```powershell 
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

## Use
- right click on one JPG or select multiple first
- go to "show more options"
- click on QuickMetadata

- Coordinates and Date/Time can be copied together by double clicking on the respective Lable
- All values can be copied by double clicking on the respective textbox
