# QuickMetadata
Small program to get specific metadata from image files.


## Installation
- Download the latest Release_v*.zip
- Unzip the program folder
- Run install.ps1 as administrator


Note: 
If Windows blocks the .ps1 file, run this command once in the PowerShell console as administrator:

```powershell
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

## Use
- Right-click on a folder, a JPG file or select multiple files first
- Go to "Show more options"
- Click on QuickMetadata
- Coordinates and date/time can be copied together by double-clicking on the respective label
- All values can be copied by double-clicking on the respective textbox
